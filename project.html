MSc Project:

Tuberculosis, Pneumonia, and Healthy and Infected lung detection using Convolutional Neural Networks: VGG16, VGG19, and ResNet50.

  • Project Description:

    The main aim of this project was to achieve the following tasks:
    1. Pneumonia detection using VGG16, VGG,19 and ResNet-50 using Transfer Learning on data.
    2. Tuberculosis detection using VGG16, VGG19 and ResNet-50 building the models from scratch on data.
    3. Merging Pneumonia and Tuberculosis X-ray images as Infected from and merging all the normal data from as Healthy dataset and creating a whole new dataset for the classification of healthy and infected X-Ray images.
    4. To do the prediction on individual models for their respective tasks.
    5. To classify X-Ray images being Pneumonia, Tuberculosis or Normal by building new CNN model for classification of 3 classes and identify which model drives the best accuracy for the 3 classes classification using VGG16, VGG19 and ResNet-50.

  • Tools and Skills used:

    1. Data Visualization:
    a. Matplotlib: A Python library for creating static, animated, and interactive visualizations in Python.
    b. Seaborn: A Python library for making statistical graphics in Python.
    2. Data Preparation:
    a. NumPy: A Python library for working with large, multi-dimensional arrays and matrices.
    b. Pandas: A Python library for data manipulation and analysis.
    c. Scikit-learn: A Python library for machine learning that includes tools for data preprocessing, feature selection, and data transformation.
    3. Model Building:
    a. TensorFlow: An open-source software library for dataflow and differentiable programming across a range of tasks.
    b. Keras: A high-level neural networks API, written in Python
    4. Model Evaluation:
    a. Scikit-learn: a Python library that includes tools for evaluating the performance of machine learning models, such as cross-validation, confusion matrix, and classification report.
    b. TensorBoard: a suite of visualization tools for TensorFlow that allows you to monitor and debug training processes.

  • Results:

    Part 1: Pneumonia detection using VGG16, VGG,19 and ResNet-50 using Transfer Learning on data.

    Part 2: Tuberculosis detection using VGG16, VGG,19 and ResNet-50.

    Part 3: Normal and infected lungs detection using VGG16 VGG19 and ResNet50.

    Part 4: Classification of healthy Tuberculosis and Pneumonia using VGG16, VGG,19 and ResNet-50.

    Part 5: Predictions for all parts.
